# 俄羅斯方塊 (Tetris) 實作計畫

這是一個基於純 HTML, CSS (Vanilla) 和 JavaScript 開發的現代化網頁版俄羅斯方塊。

## User Review Required

> [!IMPORTANT]
> 關於音效的處理，我計畫直接使用瀏覽器內建的 **Web Audio API** 來合成音效（如移動、旋轉、消除、遊戲結束的聲音），這樣就不需要額外準備 mp3 檔案，且程式碼將完全純粹。
> 另外，關於控制按鍵定義：
> - **左/右方向鍵**：左右移動
> - **下方向鍵**：加速下落 (Soft Drop)
> - **空白鍵 (Space)**：直接下落到底 (Hard Drop)
> - **上方向鍵 / X鍵**：向右旋轉
> - **Z鍵**：向左旋轉
> 
> 請確認以上的音效設計與按鍵配置是否符合您的期待？如果確認沒問題，請核准這個計畫，我就會開始實作。

## Proposed Changes

### 核心與介面

#### [NEW] [index.html](./index.html)
負責主體結構。包含遊戲主畫布 (Canvas)、下一個方塊的預覽區塊、分數/等級顯示區以及按鍵說明。

#### [NEW] [style.css](./style.css)
負責現代化、具有質感的設計。會採用淺色模式 (Light Mode)，並加入一些毛玻璃效果 (Glassmorphism) 或光暈特效，讓畫面看起來更生動且高級。

### 遊戲邏輯與音效

#### [NEW] [script.js](./script.js)
包含所有遊戲邏輯與音效合成：
1. **遊戲迴圈 (Game Loop)**：使用 `requestAnimationFrame` 進行平滑的畫面更新。
2. **方塊資料結構**：定義 I, J, L, O, S, T, Z 幾種方塊的形狀與各自專屬的鮮豔顏色。
3. **操作與碰撞**：處理左右移動、下落、直接下落與左右旋轉，並確保方塊不會超出邊界或與既有方塊重疊。
4. **消除與計分**：判斷整行填滿時進行消除，並播放消除特效與音效，同時更新分數與等級（隨著等級提升加快下落速度）。
5. **Web Audio API**：實作簡單的音效合成器 (Synth)，產生嗶嗶聲等復古遊戲音效。

## Verification Plan

### Manual Verification
- 在瀏覽器中開啟 `index.html`。
- 確認遊戲畫面是否精美且有預覽下一個方塊的功能。
- 測試所有的方向鍵與 Z/X/空白鍵是否如預期運作。
- 確認進行各項操作（移動、旋轉、消除、直接下落）時是否都有對應的音效。
- 測試方塊是否會隨機生成多種顏色且形狀正確。
